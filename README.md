Computer Organization and Architecture (CSBB251) Lab Repository


This repository contains a collection of Computer Organization and Architecture (COA) laboratory experiments, implemented and documented as part of academic coursework.


The Experiment List is as Follows 
Experiment 01 – Basic Logic Circuits Implementation of fundamental logic gates and combinational circuits 
Experiment 02 – 4-bit Ripple Carry Adder Design and simulation of a 4-bit adder using basic building blocks 
Experiment 03 – GDB Debugger (Compiler Debugging) Debugging C programs using GNU Debugger (GDB) 
Experiment 04 – 8-bit Shift Register using Flip-Flops Design a 8-bit Shift Register using D Flip 
Experiment 05 – Addressing Modes Executing different kinds of Addressing modes using gdb compiler 
Experiment 06 – Shift Registors Executing different kinds of Shify registors using logisim simulator
